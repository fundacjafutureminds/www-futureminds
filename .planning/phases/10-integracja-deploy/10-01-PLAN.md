Stworzyc nowy plik `src/components/ui/FadeIn.tsx` z dyrektywa `"use client"`.

Eksportowac 3 komponenty:

1. **FadeIn** — wrapper animujacy dzieci z fadeIn scroll-triggered:
   - Props: `children`, `direction` (FadeDirection = "up" | "down" | "left" | "right" | "none"), `delay` (number, default 0), `duration` (number, default 0.6), `className` (string opcjonalny)
   - Uzyc `motion.div` z:
     - `initial`: opacity 0 + offset wg direction (left: x=-30, right: x=30, up: y=30, down: y=-30, none: brak offsetu)
     - `whileInView`: opacity 1, x: 0, y: 0
     - `viewport`: `{ once: true, amount: 0.2 }`
     - `transition`: `{ delay, duration, ease: "easeOut" }`
   - Przekazac `className` do motion.div

2. **StaggerContainer** — kontener uruchamiajacy stagger na dzieciach:
   - Props: `children`, `staggerDelay` (number, default 0.1), `className`
   - `motion.div` z variants: hidden={}, visible={ transition: { staggerChildren: staggerDelay } }
   - `initial="hidden"`, `whileInView="visible"`, `viewport={{ once: true, amount: 0.2 }}`

3. **StaggerItem** — element wewnatrz StaggerContainer:
   - Props: `children`, `className`
   - `motion.div` z variants: hidden={ opacity: 0, y: 20 }, visible={ opacity: 1, y: 0, transition: { duration: 0.5 } }

Import: `import { motion } from "framer-motion"` (framer-motion 12.31.0 juz zainstalowane).
Typy: `import type { ReactNode } from "react"`.

WAZNE: NIE konwertowac sekcji na Client Components. FadeIn.tsx to jedyna granica "use client".
